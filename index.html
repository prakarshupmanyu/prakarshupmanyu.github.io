Abstractive French article summarization
  • An encoder-decoder system that creates an abstract summary of a French news article. It was trained using 446,000 French news articles scraped from websites like www.lemonde.fr, www.latribune.fr, etc.
  • The model was trained using Recurrent Neural Networks with LSTM units.
  • The generated summary received a FRESA score of 0.08-0.21. Further details can be found here.
Mini Prolog
  • This is a tool that compiles sentences in First Order Logic. I wrote a compiler using PLY library in Python. The processed statements were added to a knowledge base for information extraction.
  • Using the knowledge base, the tool could tell us whether a new statements is inferable or not. It gives answer in True/False.
  • The system had a 96% accuracy to infer new sentences that were humanly inferable from any given knowledge base.
